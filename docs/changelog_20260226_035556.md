## Overview

Added a manual sync button to the sidebar header so users can trigger a data sync from `~/.claude/` directly in the web UI without using the API.

## Changes

### Added: Sync button in sidebar header

A refresh icon button appears in the sidebar header between the "clarc" logo and the sidebar collapse button. Clicking it:
- Calls `POST /api/sync` to sync data from `~/.claude/` and reindex
- Spins the icon during the sync operation
- Disables the button to prevent double-clicks
- Refetches the project list on completion so counts update immediately

### Updated: IconProps supports `style` prop

Added `style?: React.CSSProperties` to the shared `IconProps` interface and the base `Icon` component, enabling inline styles (used for the spin animation) on any icon.

### Updated: Docs

- **USER_GUIDE.md**: Updated "How do I refresh the data?" FAQ to mention the sync button as the primary manual refresh method
- **DEV_GUIDE.md**: Updated `Sidebar.tsx` component description to include the sync button
